Privacy Policy

Important: This application is designed for internal team use only and does not collect personal information from external users.

1. Information We Collect

1.1 TikTok Public Data

  • Video metadata (titles, descriptions, hashtags)
  • Public engagement metrics (views, likes, comments, shares)
  • Trending hashtags and music information
  • Creator account names (public usernames only)
  • Publication timestamps and video durations

1.2 Usage Data

  • Search queries and analysis parameters
  • Application usage logs for system maintenance
  • Error logs for debugging purposes

1.3 What We Do NOT Collect

  • Personal identifying information of TikTok users
  • Private messages or personal communications
  • Device information or location data
  • Authentication credentials or passwords
  • Content from private accounts

2. How We Use Information

  • Trend Analysis: Identify emerging trends in content and hashtags
  • Research Purposes: Conduct internal market research and analysis
  • Reporting: Generate internal reports for team decision-making
  • System Improvement: Enhance application functionality and performance
  • Compliance: Ensure adherence to platform policies and regulations

3. Data Storage and Security

3.1 Storage Infrastructure

  • Data stored in secure, encrypted cloud environments
  • Industry-standard encryption for data at rest and in transit
  • Regular security audits and vulnerability assessments
  • Access controls and authentication mechanisms

3.2 Data Retention

  • Trend data retained for maximum 12 months
  • Aggregated analytics data retained for 24 months
  • System logs retained for 90 days
  • Automatic deletion after retention periods

4. Data Sharing and Disclosure

Zero External Sharing: We do not share, sell, rent, or distribute any collected data to external parties.
  • Data access limited to authorized internal team members only
  • No third-party analytics or tracking services
  • No data monetization or commercial distribution
  • Legal disclosure only if required by law

5. User Rights and Control

  • Access: Team members can request access to collected data
  • Correction: Request correction of inaccurate information
  • Deletion: Request deletion of specific data sets
  • Opt-out: Discontinue use of the application at any time
  • Transparency: Regular updates on data collection practices

6. Compliance and Legal Framework

  • GDPR compliance for EU data subjects
  • CCPA compliance for California residents
  • Adherence to TikTok's API terms of service
  • Regular legal compliance reviews
  • Data protection impact assessments

7. Changes to This Policy

We may update this privacy policy periodically. Significant changes will be communicated to all team members, and continued use of the application constitutes acceptance of the updated policy.

隐私政策

重要说明： 本应用专为内部团队使用设计，不收集外部用户的个人信息。

1. 我们收集的信息

1.1 TikTok 公开数据

  • 视频元数据（标题、描述、话题标签）
  • 公开互动指标（浏览量、点赞数、评论数、分享数）
  • 热门话题标签和音乐信息
  • 创作者账户名称（仅公开用户名）
  • 发布时间戳和视频时长

1.2 使用数据

  • 搜索查询和分析参数
  • 用于系统维护的应用使用日志
  • 用于调试的错误日志

1.3 我们不收集的内容

  • TikTok 用户的个人身份信息
  • 私人消息或个人通信
  • 设备信息或位置数据
  • 身份验证凭据或密码
  • 来自私人账户的内容

2. 我们如何使用信息

  • 趋势分析： 识别内容和话题标签的新兴趋势
  • 研究目的： 进行内部市场研究和分析
  • 报告生成： 为团队决策生成内部报告
  • 系统改进： 增强应用功能和性能
  • 合规性： 确保遵守平台政策和法规

3. 数据存储和安全

3.1 存储基础设施

  • 数据存储在安全的加密云环境中
  • 对静态和传输中的数据采用行业标准加密
  • 定期安全审计和漏洞评估
  • 访问控制和身份验证机制

3.2 数据保留

  • 趋势数据最多保留 12 个月
  • 聚合分析数据保留 24 个月
  • 系统日志保留 90 天
  • 保留期后自动删除

4. 数据共享和披露

零外部共享： 我们不会向外部方共享、出售、租赁或分发任何收集的数据。
  • 数据访问仅限于授权的内部团队成员
  • 无第三方分析或跟踪服务
  • 无数据货币化或商业分发
  • 仅在法律要求时才会披露

5. 用户权利和控制

  • 访问权： 团队成员可请求访问收集的数据
  • 更正权： 请求更正不准确的信息
  • 删除权： 请求删除特定数据集
  • 退出权： 随时停止使用应用程序
  • 透明度： 定期更新数据收集实践

6. 合规性和法律框架

  • 针对欧盟数据主体的 GDPR 合规
  • 针对加州居民的 CCPA 合规
  • 遵守 TikTok API 服务条款
  • 定期法律合规审查
  • 数据保护影响评估

7. 政策变更

我们可能会定期更新此隐私政策。重大变更将通知所有团队成员，继续使用应用程序即表示接受更新的政策。

プライバシーポリシー

重要事項： 本アプリケーションは内部チーム使用専用に設計されており、外部ユーザーの個人情報は収集しません。

1. 収集する情報

1.1 TikTok 公開データ

  • 動画メタデータ（タイトル、説明、ハッシュタグ）
  • 公開エンゲージメント指標（再生回数、いいね数、コメント数、シェア数）
  • トレンドハッシュタグと音楽情報
  • クリエイターアカウント名（公開ユーザー名のみ）
  • 公開タイムスタンプと動画時間

1.2 使用データ

  • 検索クエリと分析パラメータ
  • システムメンテナンス用のアプリケーション使用ログ
  • デバッグ目的のエラーログ

1.3 収集しないもの

  • TikTok ユーザーの個人識別情報
  • プライベートメッセージや個人通信
  • デバイス情報や位置データ
  • 認証資格情報やパスワード
  • プライベートアカウントからのコンテンツ

2. 情報の使用方法

  • トレンド分析： コンテンツとハッシュタグの新興トレンドを特定
  • 研究目的： 内部市場調査と分析の実施
  • レポート作成： チーム意思決定のための内部レポート生成
  • システム改善： アプリケーション機能とパフォーマンスの向上
  • コンプライアンス： プラットフォームポリシーと規制の遵守確保

3. データストレージとセキュリティ

3.1 ストレージインフラストラクチャ

  • 安全で暗号化されたクラウド環境でのデータ保存
  • 保存時および転送時のデータに対する業界標準の暗号化
  • 定期的なセキュリティ監査と脆弱性評価
  • アクセス制御と認証メカニズム

3.2 データ保持

  • トレンドデータは最大12ヶ月間保持
  • 集約分析データは24ヶ月間保持
  • システムログは90日間保持
  • 保持期間後の自動削除

4. データ共有と開示

外部共有ゼロ： 収集したデータを外部の当事者と共有、販売、賃貸、または配布することはありません。
  • データアクセスは認可された内部チームメンバーのみに制限
  • 第三者分析やトラッキングサービスなし
  • データの収益化や商業配布なし
  • 法的要求がある場合のみ開示

5. ユーザーの権利と制御

  • アクセス権： チームメンバーは収集されたデータへのアクセスを要求可能
  • 訂正権： 不正確な情報の訂正を要求
  • 削除権： 特定のデータセットの削除を要求
  • オプトアウト： いつでもアプリケーションの使用を中止
  • 透明性： データ収集実践の定期的な更新

6. コンプライアンスと法的フレームワーク

  • EU データ主体のための GDPR コンプライアンス
  • カリフォルニア州居住者のための CCPA コンプライアンス
  • TikTok API 利用規約の遵守
  • 定期的な法的コンプライアンス審査
  • データ保護影響評価

7. ポリシーの変更

このプライバシーポリシーは定期的に更新される場合があります。重要な変更はすべてのチームメンバーに通知され、アプリケーションの継続使用は更新されたポリシーへの同意を意味します。
